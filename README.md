# sveltekit-demo

## Sample SvelteKit demo app using Svelte.

## About the app

- The app demonstrates:

- ### Zone times

  - View time of selected zone
  - View a custom time by selecting a custom input time

- ### Dynamic data - to view various API data from [https://jsonplaceholder.typicode.com](https://jsonplaceholder.typicode.com)

    This uses inner layout and router outlet

- ### Calendars - core feature of the app

  - **Monthly**  - Regular calendar with year navigation  `[-20] [-10] [Today] [+10] [+20]`
  - **Radial** - Original circular visualization (my invention)
  - **Yearly** - Complete annual overview - with year navigation

- ### Svelte-features - some of the features shown:

  - Stores demo
  - `svelte-component` toggle
  - Checkbox selection
